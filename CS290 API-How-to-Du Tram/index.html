What's an API?


The traditional boring definition

API is an interface or platform that enables program that faciliates interation with other software program. In the context of food, an API refers to the interface or platform that enables a chef to connect with the "markets" and "recipes". APIs have multiple features that facilitate "recipes" sharing. For instances, the Yelp API let us search for restaurants that we want to explore, whereas Spoonacular API let us search for recipes.

Currenly, Yelp Fusion has two different versions:
  • V2.0 which is older and will be discontinued in April 1, 2017 :(...
  • V3.0 that is newer and easier to use.
